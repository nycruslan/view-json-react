# JsonViewer

A lightweight, performant React component for rendering JSON data with collapsible nodes, theme support, and copy-to-clipboard functionality.

## Features

- 🎨 **Light & Dark Theme** - Built-in theme support with CSS variables
- 📦 **Collapsible Nodes** - Interactive expand/collapse for objects and arrays
- 📋 **Copy to Clipboard** - Easily copy JSON values with visual feedback
- 🎯 **Type-Safe** - Written in TypeScript with full type definitions
- ⚡ **Lightweight** - ~5KB (brotli compressed)
- ♿ **Accessible** - Keyboard navigation and ARIA labels
- 🎨 **Customizable** - CSS variables and className for easy theming
- 📱 **Mobile-Friendly** - Touch device support with accessible controls
- 🔧 **Flexible API** - Root naming, expand depth control, copy callbacks
- 🌳 **Tree-Shakeable** - ESM build with proper side-effects declaration

## Installation

```bash
npm install view-json-react
```

```bash
pnpm add view-json-react
```

## Basic Usage

```tsx
import { JsonViewer } from 'view-json-react';

const data = {
  name: 'John Doe',
  age: 30,
  email: 'john.doe@example.com',
};

function App() {
  return <JsonViewer data={data} />;
}
```

## Theming

You can customize the appearance using CSS variables:

```css
.json-viewer {
  --json-bg: #ffffff;
  --json-text: #24292f;
  --json-key: #0969da;
  --json-string: #0a3069;
  --json-number: #0550ae;
  --json-boolean: #8250df;
  --json-null: #57606a;
  --json-bracket: #6e7781;
  --json-hover-bg: #f6f8fa;
  --json-border: #d0d7de;
}
```

## TypeScript

The component is fully typed and exports all necessary TypeScript definitions:

```tsx
import type { JsonValue, JsonViewerProps } from 'view-json-react';

const data: JsonValue = {
  foo: 'bar',
  nested: { count: 42 },
};

const MyComponent = () => <JsonViewer data={data} theme="dark" />;
```

## Performance

- **Bundle Size**: ~4.8KB (brotli), ~10KB (gzip)
- **Zero Dependencies**: No runtime dependencies (React is a peer dependency)
- **Tree-Shakeable**: ESM build with proper side-effects declaration
- **Optimized Rendering**: Efficient re-renders with proper memoization

## Accessibility

- Keyboard navigation for collapsible nodes (Space/Enter to toggle)
- Proper ARIA labels for screen readers
- Semantic HTML structure
- Focus indicators for interactive elements

## Browser Support

Works in all modern browsers that support:
- CSS custom properties (CSS variables)
- ES2015+ JavaScript
- React 18+
